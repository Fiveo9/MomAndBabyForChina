# VitePress 表格插件

这是一个为 VitePress 开发的表格增强插件，会自动为所有 Markdown 表格添加高级表格功能。

## 功能特性

- 🔧 **自动增强**：所有 Markdown 表格都会自动添加工具栏
- 📌 **固定表头**：支持表格表头固定
- 📍 **固定首列**：支持通过表格上方复选框开启/关闭首列固定
- ✅ **运行时控制**：固定首行和固定首列可同时勾选
- 📤 **导出表格**：支持将当前表格导出为 CSV，图片/截图会转换为文本引用
- 📐 **宽表优化**：多列表格按内容自然撑开，并在容器内横向滚动
- 🎨 **样式集成**：与 VitePress 主题完美集成

## 语法说明

直接使用标准 Markdown 表格即可，不需要添加 `:table:(...)` 前置标识。

```markdown
| 表头1 | 表头2 | 表头3 |
|-------|-------|-------|
| 内容1 | 内容2 | 内容3 |
```

## 使用示例

### 基础表格

```markdown
| 姓名 | 年龄 | 职业 |
|------|------|------|
| 张三 | 25 | 工程师 |
| 李四 | 30 | 设计师 |
```

### 多列表格

```markdown
| 品牌 | 型号 | 功能 | 价格 | 评分 | 操作 |
|------|------|------|------|------|------|
| 品牌A | 型号1 | 功能描述 | ¥1000 | 4.5 | 查看 |
| 品牌B | 型号2 | 功能描述 | ¥1200 | 4.8 | 查看 |
```

## 生成的 HTML 结构

插件会为表格添加工具栏和以下 class：

- **固定表头**：`table__fixed-header` class
- **固定首列**：`table__fixed-first-column` class
- **导出按钮**：`data-table-plugin-action="export-csv"`
- **滚动容器**：`table-plugin__scroll`

## CSS 样式

插件包含完整的 CSS 样式，支持：

- 固定表头的粘性定位
- 固定首列的粘性定位和层级管理
- 宽表横向滚动，避免多列被压缩
- 响应式设计
- 自定义滚动条样式
- 与 VitePress 主题变量集成

## 安装和配置

1. 将插件文件放置在 `.vitepress/plugins/` 目录下
2. 在 `.vitepress/config.js` 中引入并使用插件：

```javascript
import tablePlugin from './plugins/table-plugin.js';

export default defineConfig({
  // ... 其他配置
  markdown: {
    config: (md) => {
      md.use(tablePlugin);
    },
  },
});
```

3. 确保 CSS 样式已添加到主题中

## 注意事项

1. 插件会增强所有 Markdown 表格。
2. 固定首行和固定首列默认关闭，由读者在表格工具栏中开启。
3. CSV 导出会将图片/截图转换为 `[截图: 名称] 图片地址` 形式，避免单元格导出为空。
