## 一页时间轴（上海常见节奏）

| 孕周（约） | 关键词 | 对应页面 | 备注（上海常见做法） |
| --- | --- | --- | --- |
| 备孕～确诊 | 叶酸、停经、验孕 | P01 | 确诊后尽早正规医院确认宫内孕 |
| 6～8 周 | 早孕初诊、B 超 | P01 / P02 | 社区「小卡」与医院「大卡」名称不同 |
| ≤12～13 周 | 建小卡 | P02 | 社区卫生服务中心；部分要居委孕情登记 |
| 约 12 周起 | 建大卡 | P02 | 选定助产医院；热门院区早约 |
| 11～13⁺⁶ | NT | P02 | 需预约，窗口窄 |
| 14～20⁺⁶ | 唐筛 / NIPT | P02 | NIPT 自费为主；高危路径听医生 |
| 20～24 | 大排畸（系统超声） | P02 | 强烈建议按医院窗口预约 |
| 24～28 | 糖耐（OGTT）等 | P02 / P03 | 空腹、按时到检 |
| 28 周后 | 产检加密、胎动 | P05 / P04 | 固定产检点；胎动异常低阈值就医 |
| 36 周前后 | 分娩评估、NST 等 | P05 + 00～02 | 同步备齐证件与待产包 |
| 临产/剖宫产日 | 入院 | 00～02 | 跟产房/病区清单走 |

## 上海办事入口（原则）

| 事项 | 建议渠道 | 说明 |
| --- | --- | --- |
| 孕产妇健康手册 / 建小卡 | 户籍或居住地**社区卫生服务中心** | 材料与时段见 [P02](/P02-建档与产检时间表)；先电话确认 |
| 建大卡 / 系统产检 | 选定助产医院产科 | 公众号/小程序/预约台；多数需小卡 |
| 预约挂号 | 医院公众号、支付宝、随申办相关入口、「上海医联」等 | 名称与入口常改，以医院公示为准 ⏳ |
| 产前筛查与诊断知情 | 建档医院产前诊断中心 | NT/唐筛/NIPT/羊穿路径见 P02 |
| 生育保险与报销 | 单位医保 + 医院窗口 | 见 [80-育儿优惠政策](/80-育儿优惠政策) |
| 出生医学证明 / 落户 | 分娩医院 + 公安 | 见 [00-待产证件资料](/00-待产证件资料) |
