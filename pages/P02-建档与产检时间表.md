# 建档与产检时间表

> **地域说明**：正文默认按**上海市**建档与产检常见节奏整理，**必须以建档医院书面计划为准**。
>
> **本页优先级最高**（实操最密）。
>
> **状态**：空壳大纲；检查节点为国内常见节奏 + 上海备注位。

## 1. 名词对照（上海口语）

| 说法 | 通常指 | 待核实备注 |
| --- | --- | --- |
| 建册 / 发手册 | 孕产妇保健手册、社区或医院录入 | ⏳ |
| 建档 / 建卡 | 选定产检+分娩医院，纳入该院产检路径 | ⏳ 名额、户籍/居住证政策因院而异 |
| 产检点 | 固定医院或联合体路径 | ⏳ 可否中途转诊 |

## 2. 建档前准备（大纲）

### 2.2 选医院思路（大纲）

- [ ] 家近 vs 医疗实力 vs 床位/建档难度

- [ ] 高危妊娠：先评估再选点
- [ ] 预约渠道：随申办 / 医院小程序 / 现场（待填）

### 2.3 建档当日流程（大纲）

1. 预约成功 → 2. 取号/报到 → 3. 建档资料审核 → 4. 抽血/ auxiliary → 5. 医生评估与产检计划  
⏳ 补：常见耗时、空腹要求、陪同人数
