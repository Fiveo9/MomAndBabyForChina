# 营养、体重与补充剂

> **地域说明**：膳食举例会带一点上海生活场景，原则全国通用；补充剂剂量**不替代医嘱**。
>
> **阶段**：全孕期。
>
> **状态**：空壳大纲。

## 1. 体重管理（大纲）

| 孕前 BMI 区间 | 建议增重总范围（待按权威指南填） | 每周增重示意 | 备注 |
| --- | --- | --- | --- |
| 偏低 | ⏳ | ⏳ | |
| 正常 | ⏳ | ⏳ | |
| 超重 / 肥胖 | ⏳ | ⏳ | 需个体化 |

### 待写

- [ ] 怎么称、何时称、孕吐期怎么看体重
- [ ] 增重过快 / 过慢：先排查再节食或猛补

## 2. 膳食原则（大纲）

| 类别 | 建议方向 | 慎用/避免（示例骨架） |
| --- | --- | --- |
| 主食 | 粗细搭配 | |
| 蛋白 | 鱼禽蛋瘦肉豆 | 生鱼片、未全熟蛋等 |
| 蔬果 | 多样清洗 | 未洗净芽菜等 |
| 奶及奶制品 | | 未灭菌乳 |
| 油脂坚果 | | 霉变 |
| 饮品 | 足量饮水 | 酒精、过量咖啡因 |

### 上海生活化备注位

- [ ] 外卖/便利店怎么选相对稳妥
- [ ] 大闸蟹、醉虾蟹、生腌：明确「不建议」表述
- [ ] 节假日聚餐策略

## 3. 补充剂地图（大纲）

| 补充剂 | 常见讨论点 | 谁更可能需要 | 注意 |
| --- | --- | --- | --- |
| 叶酸 / 含叶酸复合 | 孕早期重点 | 几乎全员评估 | 剂量听医嘱 |
| 铁剂 | 贫血化验后 | 化验异常 | 便秘等副作用 |
| 钙 | 膳食不足时 | | 与铁间隔 |
| 维生素 D | 化验/日照 | | |
| DHA | 膳食来源 vs 补充 | | 选鱼油注意汞与来源 |
| 碘 | 加碘盐与海产品 | | 甲状腺疾病个体化 |

### 待写原则

- [ ] 「能吃饭优先于堆保健品」
- [ ] 网红补剂避雷清单（空列）
- [ ] 与慢性病用药冲突时找谁问（产科门诊 / 临床药学）

## 4. 特殊情形（大纲）

- [ ] 妊娠剧吐
- [ ] 妊娠期糖尿病饮食（仅框架，细节跟营养门诊）
- [ ] 素食 / 过敏体质
- [ ] 双胎
