## 1. 孕晚期产检变化

| 主题 | 常见变化 | 你要配合的 |
| --- | --- | --- |
| 产检频率 | 多从 4 周一次 → 2 周一次 → 足月后每周（因人而异） | 按预约来；高危更密 |
| 血压 / 尿蛋白 | 子痫前期筛查意识增强 | 有头痛视物模糊上腹痛按 [P04](/P04-常见不适与红旗信号) 急诊 |
| 血常规等 | 查贫血、感染等 | 贫血治疗别自行停药 |
| 超声 | 评估胎位、生长、羊水、胎盘位置等 | 按医院节点约，勿拖到无号 |
| 胎心监护（NST） | 常自 36 周前后开始或有指征更早 | 餐后、左侧卧，留足时间 |
| **GBS** 等 | 约 35～37 周常见（以医院为准） | 阳性则临产抗生素策略听医生 |
| 胎位 | 臀位等有处理时间窗 | 外倒转等**仅在有指征且医院具备条件时**讨论 |

更完整节点表见 [P02](/P02-建档与产检时间表)。

## 2. 胎动

### 2.1 原则

- **从何时开始规律计数**：听建档医生教学（常见自孕中晚期有固定胎动模式后）。  
- 方法：市面有多种计数法（如固定时段内感受次数等）——**以本院宣教为准**，本页不写死单一学派。  
- 关键的是：**熟悉自己宝宝的日常模式**；突然变少、变弱、模式乱要警惕。

### 2.2 感觉减少时的行动路径

1. 左侧卧，安静环境，避开玩手机分心  
2. 按医生教过的方法再评估一段时间  
3. **仍明显减少或消失 → 急诊/产科监护**，不要硬等明早门诊  
4. 有合并出血、腹痛、破水等，直接急诊

## 3. 分娩征兆 vs 假性宫缩

| 表现 | 可能意义 | 何时出发（原则） |
| --- | --- | --- |
| **见红** | 临近分娩可能；也可能仍有数日 | 量少可电话/按医院嘱；**量多如月经或伴规律宫缩**就医 |
| **破水** | 胎膜破裂 | 记时间颜色气味；多数应**及时入院**，尤其早产孕周 |
| **规律宫缩** | 临产可能 | 间隔缩短、强度增加、活动后不消失；具体「几分钟一阵」跟医院宣教 |
| **假性宫缩** | 子宫练习收缩 | 休息补水后减弱；无进行性增强 |
| 胎动异常 | 见上节 | 急诊 |
| 阴道流血明显增多 | 排除产前出血等 | **急诊**，勿在家观察太久 |

### 3.1 上海实操小提示

- 夜间出发预留堵车与停车时间；导航备**建档医院急诊/产科入口**  
- 手机电量、就诊卡/医保码、小卡与产检资料放固定袋  
- 来不及回建档医院时，就近有产房的医院优先保母婴安全，再协调转诊  

## 4. 分娩方式知情（沟通提纲）

| 方式 | 适合讨论的点 | 恢复粗线条 | 待产准备 |
| --- | --- | --- | --- |
| **阴道分娩** | 产程、会阴、镇痛（无痛）指征与时机 | 下床与恶露因人而异 | 见 [01 妈妈包](/01-妈妈待产&住院包) |
| **剖宫产** | 医学指征、择期 vs 急诊、麻醉方式 | 伤口护理、排气、活动 | 收腹带等见 01；听病区 |
| **瘢痕子宫试产（VBAC 等）** | 仅部分条件合适；风险收益要当面谈 | 失败可能中转剖 | 高危路径，选有能力的医院 |

> 没有「绝对高级」的分娩方式，只有**当下母婴安全最优解**。网友案例不能当自己的手术指征。

## 6. 入院后流程（文字版）

```
急诊/产科分诊
    ↓
评估：宫缩、胎心、是否破水、宫颈等
    ↓
待产室或病区住院
    ↓
产程进展 → 产房阴道分娩
  或 手术指征 → 术前准备 → 手术室
    ↓
分娩后：母婴初步处理、回室、观察出血与宫缩
    ↓
新生儿：疫苗、听力等按院内流程
    ↓
出院宣教 → 社区访视（小卡回传相关）→ 产后页面
```

夜间话术示例（可改）：「我是 ____ 医院建档孕妇，孕 ____ 周，现在（规律宫缩/破水/出血/胎动少），从哪个门进产科急诊？」
